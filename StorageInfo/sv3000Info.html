HPE StoreVirtual 3000 Overview:

StoreVirtual 3000 Storage is an advanced storage architecture designed with simplicity in mind to offer a highly virtualized backend and a single storage capacity pool for superior efficiency. It offers a single screen to manage the platform via an intuitive GUI with a common look and feel with other Hewlett Packard Enterprise user experiences. This hardware platform runs the StoreVirtual operating system providing unique scalability, resiliency, and management capabilities to deliver a cost-effective, shared storage system for a wide range of SMB customers.

Product Overview
The HPE StoreVirtual 3000 Storage is designed for traditional hard drives with the option to transition to hybrid flash at any time for increased performance. The optional Adaptive Optimization feature reacts to real-time I/O fluctuations as workloads change to make sure the most frequently used data is automatically moved to the fastest storage media without user intervention. Offering a very affordable, granular building block approach, the StoreVirtual 3000 delivers low-entry price points with robust scale-up capabilities. Start with a base configuration and scale as needs grow, providing investment protection and extended ROI in a world where throw-away mentality is still common. Users can upgrade the StoreVirtual 3000 by adding flash to deliver application acceleration when ready.
